# PeakRun-AI
从第一公里到马拉松，PEAKRUN AI 为你制定专属训练计划，分析数据，语音陪跑，独特数据分析。

核心功能

🧠 AI 智能训练计划

根据你的能力、训练表现和目标，智能生成专属训练计划，并随着训练进度动态调整，让每一次训练都更加科学、高效。

📈 智能跑步分析

深度分析配速、心率、训练负荷等训练数据，为每一次跑步提供专业反馈和建议，帮助你持续提升表现。

⌚ Apple Watch 跑步记录

支持 Apple Watch 独立记录跑步，实时记录配速、距离、心率等数据，并自动同步至 Apple 健康。

🎧 语音陪跑

训练过程中实时播报配速、距离、心率等关键数据，让你专注跑步，无需频繁查看手表。

📊 专业训练数据

完整记录每一次训练，支持查看训练趋势、配速变化、心率曲线等关键指标，帮助你了解自己的成长轨迹。

适合人群

* 跑步新手
* 日常健身跑者

* 全程马拉松跑者
